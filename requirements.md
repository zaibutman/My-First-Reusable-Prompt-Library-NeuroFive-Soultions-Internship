# Project Requirements

## Project Title

**Build Your First Reusable Prompt Library**

---

## Overview

This project demonstrates how to create a reusable prompt template for generating professional LinkedIn captions using Large Language Models (LLMs). Instead of writing prompts from scratch, a structured template with placeholders is used to produce consistent, high-quality outputs for different topics.

The prompt template follows five core components:

- **Role**
- **Context**
- **Task**
- **Format**
- **Constraints**

By modifying only the variable values, the same template can be reused for multiple prompt instances.
